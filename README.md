# flutter
##各种问题总结，废话不多，直入主题

1、编辑器IntelliJ IDEA使用Flutter插件出现Error running 'main.dart': not implemented

打开编译器Preferences->Plugins->Installed，查看已安装的插件，试着去禁用下载的与flutter无关的插件，然后重启运行。
本人禁用的是Matlab Support插件。
